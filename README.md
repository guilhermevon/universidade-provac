# 🎓 Provac University

> An internal learning management platform (LMS) designed to deliver corporate training, organize educational content, and promote employee engagement through a gamified ranking system.

Provac University is a web-based platform developed to centralize employee learning resources, providing access to courses, lessons, and educational materials while encouraging continuous learning through a points and ranking system.

---

## 💻 Technologies

* **Node.js**
* **Express.js**
* **JavaScript**
* **Python**
* **Docker**
* **NGINX**
* **REST APIs**
* **Third-party Libraries**

## 🌐 Features

### 🎓 Course Management

Provides a centralized environment for organizing and accessing internal training courses.

### 📚 Lesson Delivery

Allows employees to follow structured learning paths through interactive lessons and educational content.

### 🏆 Gamification System

Implements a points and ranking system to encourage engagement and continuous learning.

### 👥 Employee Learning Platform

Offers a dedicated space where employees can access training materials and monitor their learning progress.

### 🔐 Secure Authentication

Provides authenticated access to ensure that only authorized users can access the platform.

### ⚡ RESTful Architecture

Built on a scalable REST architecture using Node.js and Express.

### 🐳 Containerized Deployment

Application deployment managed through Docker containers with NGINX acting as a reverse proxy.

### 🐍 Python Integrations

Uses Python scripts and libraries to support specific processing and automation tasks within the platform.

## 📝 About the Project

Provac University was developed as an internal corporate platform to support employee training and professional development.

The project was designed with scalability and maintainability in mind, combining Node.js, Express, Docker, NGINX, and Python to provide a reliable learning environment capable of managing educational content and user engagement.

Since this is a proprietary corporate application, only general information about its architecture and technologies can be shared.
